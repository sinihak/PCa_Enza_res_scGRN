### AR suppression rewires a SOX4 regulatory program to promote enzalutamide resistance in prostate cancer

#### This repository contains the code to reproduce the results and figures in the manuscript.

##### Package versions:
- **glue_gpu.yaml** and **pyscenic.yaml**: Python packages for transcription factor (TF) regulatory program construction and other downstream analyses, excluding TF footprinting
- **renv.yaml**: R packages
- **scprinter.yaml**: Python packages for TF footprinting analysis
